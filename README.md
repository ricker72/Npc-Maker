## 🎉 Creado por @ricker72

**NPC Maker Pro** es una herramienta **open‑source** de primera calidad para diseñar y gestionar NPCs en servidores **OpenTibia / Canary / CrystalServer**. La aplicación combina un **tema oscuro moderno**, **micro‑animaciones fluidas**, **efectos de cristal** y un **diseño responsive** que brinda una experiencia visual premium.

---

## ✨ Principales Características

- **Interfaz Dinámica** – Transiciones suaves al cambiar de pestaña, arrastrar elementos y previsualizar scripts Lua.
- **Vista Previa en Vivo** – Generación de código Lua con efecto de *máquina de escribir* y resaltado de sintaxis.
- **Editor de Monstruos (Monster Editor)** –
  - Define stats, loot, habilidades y comportamiento.
  - Previsualiza el script Lua generado en tiempo real.
  - Exporta directamente a archivo `.lua` listo para usar en el servidor.
- **Creador de Scripts (Script Creator)** –
  - Editor de scripts de misiones con resaltado y autocompletado.
  - Soporte para bloques de texto y plantillas de misión.
  - Vista previa instantánea del script generado.
- **Diseñador de Apariencia** – Selector de outfits con rotación 3D y animaciones de color.
- **Constructor de Tienda** – Lista de ítems arrastrables con animaciones de rebote.
- **Exportación / Importación** – Un clic para exportar a Lua o JSON, con barra de progreso animada.
- **Integración con el Agente IA RME** – Generación automática de mapas y misiones mediante IA.

## 📦 Instalación

1. Descarga el instalador **NPC Maker Pro Setup 2.0.0.exe** desde la [página de releases](https://github.com/ricker72/npc-maker/releases).
2. Ejecuta el instalador y **espera a que finalice** (verás una barra de progreso con partículas animadas).
3. Una vez completado, inicia **NPC Maker Pro** desde el menú de inicio o el acceso directo del escritorio.

> 🛠️ No necesitas comandos adicionales: el instalador gestiona todas las dependencias automáticamente.
