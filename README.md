## 🚀 Overview

**NPC Maker Pro** is a premium, open‑source tool for designing non‑player characters (NPCs) for the **Canary Tibia** server. The interface features a modern dark theme with smooth micro‑animations, glass‑morphism panels and responsive layout, making NPC creation an engaging visual experience.

---

## ✨ Highlights & Animations

- 🎨 **Dynamic UI** – Animated transitions when switching tabs, dragging elements, and previewing Lua scripts.
- ⚡ **Live Lua Preview** – Real‑time syntax‑highlighted Lua generation with typing animations.
- 📦 **Export/Import** – One‑click export to a ready‑to‑use Lua file or JSON configuration, with animated progress bars.
- 🛠️ **Outfit Designer** – Visual outfit customization with animated colour pickers and preview rotators.
- 🛒 **Shop Builder** – Drag‑and‑drop item list with smooth sliding animations.
- 📚 **Keyword‑Based Dialogues** – Instant keyword addition with fade‑in feedback.

---

## 📦 Installation

1. Download the **NPC Maker Pro Setup 2.0.0.exe** from the release page.
2. Run the installer and **wait until the installation finishes**.
3. Once the setup completes, launch **NPC Maker Pro** from the start menu or desktop shortcut.

> No additional commands are required – the installer handles all dependencies.
